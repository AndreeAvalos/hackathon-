## Comando basicos

`dotnet restore`
- Restaurar base de datos
`dotnet ef database update --context=CustomContext`
- Compilar solucion
`dotnet build`
- Ejecutar aplicacion
`dotnet run`
- Ejecutar pruebas unitarias
`dotnet test`
